# Daily Worship Guide v2 | دليل العبادة اليومية

A complete Islamic daily worship schedule with full customization.

## Features
- All 7 default prayer blocks (Tahajjud → Isha) with original acts preserved
- Checkboxes to track daily completion
- Daily progress bar
- Motivational hadith per block
- Add your own worship acts to any block
- Edit or delete existing acts
- Create entirely new time blocks
- Set custom prayer times
- Everything saved permanently in browser (localStorage)

## 🚀 Deploy with GitHub + Vercel (Recommended)

### Step 1 — Push to GitHub
```bash
# Unzip and enter the folder
unzip daily-worship-guide-v2.zip
cd daily-worship-guide-v2

# Initialize git
git init
git add .
git commit -m "Daily Worship Guide v2"

# Create repo on github.com then connect:
git remote add origin https://github.com/YOUR_USERNAME/daily-worship-guide.git
git branch -M main
git push -u origin main
```

### Step 2 — Deploy on Vercel
1. Go to **vercel.com** → Sign up with GitHub
2. Click **Add New Project**
3. Select your `daily-worship-guide` repo
4. Click **Deploy** — done! 🎉

Your app will be live at:
`https://daily-worship-guide.vercel.app`

### Step 3 — Future Updates
Every time you push to GitHub, Vercel auto-deploys:
```bash
git add .
git commit -m "your update"
git push
```
